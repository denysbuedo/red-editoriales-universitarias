Red Nacional de Editoriales Universitarias (RNEU)

Una Red de Editoriales · Un Catálogo Nacional · Una Plataforma para el Conocimiento

Resumen Ejecutivo

Este documento propone evolucionar la idea inicial de un portal institucional hacia una infraestructura digital nacional para la difusión de la producción editorial universitaria cubana.

La propuesta no sustituye a las editoriales ni centraliza su gestión. Construye una plataforma federada donde cada institución conserva su identidad y sus sistemas, mientras comparte un catálogo nacional común.

1. Contexto

Las editoriales universitarias cubanas generan una parte significativa del patrimonio científico, académico y cultural del país. Sin embargo, esa producción permanece distribuida entre múltiples sitios institucionales, repositorios y catálogos independientes.

La consecuencia es una baja visibilidad nacional e internacional, dificultades para localizar publicaciones y ausencia de un punto único de descubrimiento.

El problema principal no es tecnológico. El problema es la fragmentación del ecosistema editorial.

2. Visión Estratégica

Crear la Plataforma Nacional de Publicaciones Universitarias de Cuba (PNPU), concebida como una infraestructura digital que permita descubrir, preservar e integrar la producción editorial universitaria mediante estándares abiertos e interoperabilidad.

4. Arquitectura Conceptual

Sistema de Gestión de Editoriales │ │ REST API ▼ Portal Nacional RNEU ▲ │ API Omeka S │ ▼ Libros · Autores Colecciones

El Portal no será propietario de la información institucional ni bibliográfica. Actuará como capa de integración, búsqueda y presentación.

5. Componentes de la Plataforma

  • Portal RNEU: experiencia pública y descubrimiento.
  • Sistema de Gestión de Editoriales: fuente oficial de datos institucionales.
  • Omeka S: gestión bibliográfica y metadatos.
  • Motor de búsqueda: indexación nacional.
  • API pública: interoperabilidad con otros sistemas.

6. Beneficios

  • Mayor visibilidad internacional.
  • Catálogo nacional único.
  • Fortalecimiento de la marca RNEU.
  • Mejor aprovechamiento del patrimonio editorial.
  • Reducción de duplicidades.
  • Base para futuras integraciones con Google Scholar, Crossref, OAI-PMH e IIIF.

Conclusión

La Red Nacional de Editoriales Universitarias no debe entenderse únicamente como una organización de coordinación. Debe convertirse en la marca que articula una infraestructura nacional para la difusión del conocimiento universitario cubano.

Este documento constituye el punto de partida para el diseño de la Plataforma Nacional de Publicaciones Universitarias de Cuba y servirá como base para los documentos de arquitectura de software, modelo de datos, APIs e infraestructura.
